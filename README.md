# Diabetes Prediction Web App (SVM + LIME)

This is a machine learning web app that predicts whether a patient is diabetic based on input features.

Model: Support Vector Machine (SVM)  
Explainability: LIME  
Frontend: Streamlit  

Users can input patient data and get:
- Prediction (Diabetic / Not Diabetic)
- Feature contribution explanation using LIME

Deployed using Streamlit Community Cloud.
